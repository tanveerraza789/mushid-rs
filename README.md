# mushid-rs

rust implementation of [mushid](https://github.com/kehiy/mushid)

## Installation

Install mushid with cargo

```bash
cargo add mushid
```

or use with git

```toml
[dependency]
mushid = { git = "https://github.com/tanveerraza789/mushid-rs.git" }
```

## Usage/Examples

```rust
use mushid::mushid;

let id = mushid();
```
