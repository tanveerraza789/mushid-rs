# mushis-rs

rust Implementation of [mushid](https://github.com/kehiy/mushid)

## Installation

Install mushid-rs with cargo

```bash
cargo add mushid
```

## Usage/Examples

```rust
use mushid::mushid;

let id = mushid();
```
